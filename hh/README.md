# 五子棋微信小程序

一个简单的五子棋小游戏，支持单人（与AI对战）和双人模式。

## 功能特点

- 支持单人模式（与AI对战）
- 支持双人模式（两个玩家轮流下棋）
- 自动判断胜负
- 显示最后一步棋的位置
- 记录游戏步数

## 如何使用

1. 使用微信开发者工具打开项目
2. 在首页选择游戏模式（单人/双人）
3. 点击"开始游戏"进入游戏页面
4. 点击棋盘位置落子
5. 游戏结束后可以点击"重新开始"或"返回主页"

## AI策略

单人模式下，AI使用以下策略（按优先级排序）：
1. 如果能形成五子连珠获胜，则选择该位置
2. 如果能阻止对手形成五子连珠，则选择该位置
3. 尝试形成活四（两端都开放的四子连珠）
4. 阻止对手形成活三
5. 尝试形成活三和其他有利局面
6. 优先选择靠近棋盘中心的位置
7. 没有明显最优选择时随机落子

每个位置的评分基于：
- 能形成的连珠数量
- 连珠的开放端数量（活棋/死棋）
- 与棋盘中心的距离
- 进攻与防守的平衡
